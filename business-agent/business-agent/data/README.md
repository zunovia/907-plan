# データディレクトリ

このフォルダに作業対象のファイルを配置してください。

## 配置するファイル

### TKCデータ（必須）
- `全仕訳明細XXXXXXXX.XLSX` — TKCから切り出した仕訳データ
- `TKＣ切り出しYYYYMM_科目残高_*.xlsx` — 科目残高データ

### 給与データ（給与仕訳生成時）
- `給与ﾃﾞｰﾀ_支給実績_給与_.CSV` — Shift_JIS(cp932)エンコーディング
- `社員ﾃﾞｰﾀ_社員情報_基本_.CSV` — 社員マスタ

### 取引先マスタ
- `取引先名.XLSX` または `全取引先名.xlsx`

### リサイクラー活動記録処理時
- 手書き活動報告書PDF（ステーション/事業所）
- リサイクラー参加表テンプレート（Excel）

## 出力先
処理結果は `output/` サブフォルダに出力されます。
