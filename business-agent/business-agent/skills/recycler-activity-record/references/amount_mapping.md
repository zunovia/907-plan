# 金額→カテゴリ判定ルール・計算式

## 基本原則

PDFに記載された**金額**が最も信頼できる情報源。
事業所名の丸囲みや作業時間の記載より、金額から逆引きで判定すること。

## 諸謝金レート表（10月〜翌9月）

### ステーション（参加人数による固定レート — 時間帯分類なし）
| 参加人数 | 日給 |
|---|---|
| 1名 | 1,700円 |
| 2名 | 1,200円 |

ステーションは時間帯（2-3.5hr等）の概念がない。署名人数のみで日給が決まる。

### 物流センター（仕分け作業）
| 時間帯 | 日給 | レート表参照セル |
|---|---|---|
| 2-3.5時間 | 1,200円 | E5 |
| 3.5-5時間 | 1,850円 | E4 |
| 5時間以上 | 2,600円 | E6 |

### Re☆ショップ / 物流補助（あつた/さくら/よしの共通）
| 時間帯 | 日給 | レート表参照セル |
|---|---|---|
| 2-3.5時間 | 850円 | E17 |
| 3.5-5時間 | 1,350円 | E16 |
| 5時間以上 | 1,850円 | E18 |

### サブマネ（伊藤ひとみのさくらセクションのみ）
| 時間帯 | 日給 | レート表参照セル |
|---|---|---|
| 2-3.5時間 | 1,350円 | E11 |
| 3.5-5時間 | 2,350円 | E10 |
| 5時間以上 | 3,350円 | E12 |

## 金額→事業所の逆引き判定

PDFに記載された金額から事業所を特定する際の判定フロー：

1. **1,700円 or 1,200円** → ステーション（人数で区別）
2. **2,600円** → 物流仕分け（5hr+）のみ
3. **1,850円** → 物流仕分け（3.5-5hr）**または** Re☆ショップ系（5hr+）→ 丸囲み事業所名で区別
4. **1,200円** → 物流仕分け（2-3.5hr）**または** ステーション2名 → PDF種別で区別
5. **1,350円** → Re☆ショップ系（3.5-5hr）**または** サブマネ（2-3.5hr）→ 伊藤ひとみ＋さくらかどうかで区別
6. **850円** → Re☆ショップ系（2-3.5hr）のみ
7. **2,350円 / 3,350円** → サブマネのみ

## 支払額の数式パターン

```python
rate_sheet = '諸謝金額と所得税率(10月～)'

# ステーション1名（固定レート）
f"=(W{r}+X{r})*1700"

# ステーション2名（固定レート）
f"=(AO{r}+AP{r})*1200"

# 物流仕分け
f"=BW{r}*'{rate_sheet}'!$E$5+BX{r}*'{rate_sheet}'!$E$4+BY{r}*'{rate_sheet}'!$E$6"

# 物流補助（Re☆ショップ系レート）
f"=CZ{r}*'{rate_sheet}'!$E$17+DA{r}*'{rate_sheet}'!$E$16+DB{r}*'{rate_sheet}'!$E$18"

# あつた（Re☆ショップ系レート）
f"=EP{r}*'{rate_sheet}'!$E$17+EQ{r}*'{rate_sheet}'!$E$16+ER{r}*'{rate_sheet}'!$E$18"

# さくら 通常（Re☆ショップ系レート）
f"=GB{r}*'{rate_sheet}'!$E$17+GC{r}*'{rate_sheet}'!$E$16+GD{r}*'{rate_sheet}'!$E$18"

# さくら 伊藤ひとみのみ（サブマネレート）
f"=GB{r}*'{rate_sheet}'!$E$11+GC{r}*'{rate_sheet}'!$E$10+GD{r}*'{rate_sheet}'!$E$12"

# よしの（Re☆ショップ系レート）
f"=HE{r}*'{rate_sheet}'!$E$17+HF{r}*'{rate_sheet}'!$E$16+HG{r}*'{rate_sheet}'!$E$18"

# 源泉所得税
f"=INT(IA{r}*'{rate_sheet}'!$A$21/100)"
```

## 注意：金額が既知のレートに合わない場合

7〜9月は別単価体系の可能性がある。
OCRで読み取った金額がどのレートにも一致しない場合は、即座にユーザーに報告すること。
