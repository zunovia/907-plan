# テンプレート列構成（リサイクラー参加表）

## 重要: 毎月のRow-名前マッピング構築が必須

テンプレートの行構成は月によって変わる可能性がある（新規追加・削除）。
**前月のRow番号を流用せず、毎月必ずテンプレートシートから直接読み取ること。**

```python
# 毎月実行：最新のRow-名前マッピング構築
name_row_map = {}
for row in range(5, ws.max_row + 1):
    name = ws.cell(row=row, column=2).value  # B列=氏名
    if name and str(name).strip():
        name_row_map[str(name).strip()] = row
```

## 時間帯カテゴリの列配置（各セクション共通パターン）

各セクションの回数計は3列構成：
- 列1: 2-3.5時間の回数
- 列2: 3.5-5時間の回数
- 列3: 5時間以上の回数
